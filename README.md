# dave

cli for automating my dev workflows. built in pure rust.

## pre-requisites:

- [rust](https://www.rust-lang.org/tools/install)
- [gum](https://github.com/charmbracelet/gum)

## getting started:

### development:

```shell
# run linters
make ci

# test changes
make dev

# clean up
make clean
```

### install as binary:

```shell
make install
```

## usage

```shell
dave
```
