# dev-cli

cli for automating my dev workflows. built in pure rust.

## pre-requisites:

- [rust](https://www.rust-lang.org/tools/install)
- [gum](https://github.com/charmbracelet/gum)

## getting started:

### run ci tools:

```shell
make ci
```

### test changes:

```shell
make dev
```

### install as binary:

```shell
make install
```

### clean up:

```shell
make clean
```
